# Supersoft Website

Official website for Supersoft and the CrumbOS project.

## About

Supersoft is an independent software project focused on creating experimental software and operating systems.

The main project is **CrumbOS**, a custom operating system featuring:

- Custom bootloader
- Assembly-based kernel
- .cru file system
- Experimental application platform
- Open-source development

## Website Features

- Project overview
- CrumbOS information
- Download links
- Documentation access
- GitHub integration
